LineChartView
=============

LineChartView  Dynamic line     动态的折线图,后续会慢慢完善，有完整的demo 
效果类似runsastic Mountain Bike Pro软件里的折线图
动态加载折线，当到控件的右边之后将会，重新计算折线宽度，向里压缩。
添加了setMapHalve(index)此方法可以将集合点数减半。
